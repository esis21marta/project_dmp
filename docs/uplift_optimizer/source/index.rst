Welcome to Uplift Optimizer's documentation!
=========================================

Here is the documentation for the pipelines, functions and classes
that the Uplift Optimizer is composed of. The Uplift Optimizer can be
used to train uplift models for campaign-user targeting and score MSISDNs at scale.

For the user manual, please refer to these pages:

- `Model Factory <https://tsdmp.atlassian.net/wiki/spaces/IN/pages/711098411/Model+Factory+-+Step+by+step+guide+to+configure+train+a+new+campaign+model+As+of+9+Oct+2020>`_
- `Model Scoring <https://tsdmp.atlassian.net/wiki/spaces/IN/pages/726728725/Model+Scoring-+Step+by+step+guide+to+configure+score+msisdns+with+multiple+trained+models+As+of+15+Oct+2020>`_

The Uplift Optimizer is composed of the Model Factory for training models,
Model Scoring for scoring at scale and Model Classes containing different uplift model types.
